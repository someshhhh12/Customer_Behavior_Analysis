# Customer Behavior Analysis

## Overview
This project analyzes customer purchasing behavior using Python, PostgreSQL, SQL, and Power BI. The goal is to identify customer preferences, spending patterns, and purchasing trends through data analysis and visualization.

## Dataset
The dataset contains customer information such as:
- Age
- Gender
- Product Category
- Purchase Amount
- Location
- Payment Method
- Subscription Status
- Purchase Frequency

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- PostgreSQL
- SQL
- Power BI
- Gamma (Presentation)

## Project Workflow
1. Load dataset using Python.
2. Clean and preprocess the data.
3. Perform Exploratory Data Analysis (EDA).
4. Store data in PostgreSQL.
5. Run SQL queries for business insights.
6. Build interactive Power BI dashboards.
7. Create project report and presentation.

## Dashboard Insights
- Customer demographics analysis
- Spending behavior analysis
- Product category performance
- Purchase frequency trends
- Payment method preferences
- Seasonal purchasing patterns

## Results
The analysis helped identify customer buying patterns, popular product categories, and factors influencing purchasing decisions.

## How to Run
1. Clone the repository.
2. Install required Python libraries.
3. Load the dataset.
4. Execute data cleaning and EDA scripts.
5. Import data into PostgreSQL.
6. Connect Power BI to PostgreSQL and build dashboards.
